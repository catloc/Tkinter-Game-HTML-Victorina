1. "С помощью какого тега в HTML задаются ссылки?"
1. <a> - Верный ответ.
2. <hr> - Неверный ответ. Этот тег задает разделяющую линию между заголовком и основным текстом.
3. <br> - Неверный ответ. Этот тег разделяет текст, перемещая часть после него строкой ниже.
4. <h1> - Неверный ответ. Этот тег предназначен для заголовков.

2. "Как называется верхнее меню сайта?"
1. footer - Неверный ответ. Футер - это нижняя часть сайта, где написаны контакты и лицензия.
2. main - Неверный ответ. Майн - часть сайта, которая содержит в себе основной контент, например, карточки товаров.
3. head - Тег в HTML, в котором задаются основные пармаетры документа, такие как язык сайта, иконка.
4. header - Верный ответ. Верхнее меню сайта называют шапкой.

3. "Где размещён основной контент?"
1. body - Неверный ответ. В "теле" документа находятся все видимые пользователю части, такие как: хедер, майн, футер.
2. footer - Неверный ответ. В футере не распологаются карточки товаров.
3. main - Верный ответ.
4. header - Неверный ответ. В шапке располагается меню сайта.

4. "С помощью чего подключают стили документу HTML?"
1. css - Верный ответ.
2. json - Неверный ответ. Там не хранятся стили для документа.
3. py - Неверный ответ. Это запускаемый файл.
4. html - Неверный ответ. В HTML можно подключить стили, но создание ещё одного документа с этим не поможет.

5. "Какой тег отвечает за заголовки?"
1. <h1> - Верный ответ. 
2. <a> - Неверный ответ. Этот тег отвечает за ссылки.
3. &nbsp - Неверный ответ. Это не тег, а знак неразрывного пробела.
4. <div> - Неверный ответ. Этот тег используется для создания "контейнеров", в которых размещают объекты и стили.

6. "Какой тег пишется в начале документа?"
1. <head> - Неверный ответ. <head> пишется после тега <html>.
2. <html> - Верный ответ.
3. <body> - Неверный ответ. Этот тег располагается после <head>.
4. <meta> - Неверный ответ. Тег <meta> определяет метаданные о документе HTML. Метаданные - это данные (информация) о данных. Теги <meta> всегда находятся внутри элемента <head>, и обычно используются для указания набора символов, описания страницы, ключевые слова, автор документа и настройки видового экрана. Метаданные не будут отображаться на странице, но могут быть проанализированы машиной.

7. "Как прописываются классы в css?"
1. .class - Верный ответ.
2. class - Неверный ответ.
3. href - Неверный ответ. 
4. div class - Неверный ответ.

8. "С помощью чего указывается путь для изображения?"
1. alt - Неверный ответ.
2. img - Неверный ответ. С помощью этого тега мы задаем лишь само поле для изображения.
3. src - Верный ответ.
4. scr - Неверный ответ. Специально допущена ошибка, так как верно "SRC".

9. "С помощью какого тега создается маркированный список?"
1. <li> - Неверный ответ. Определяет отдельный элемент списка.
2. <ul> - Верный ответ.
3. <img> - Неверный ответ. Задаётся изображение.
4. <div> - Неверный ответ. Задаётся "контейнер".

10. "Как обозначается закрывающий тег параграфа?"
1. </div> - Неверный ответ.
2. </html> - Неверный ответ.
3. </h1> - Неверный ответ.
4. </p> - Верный ответ.
